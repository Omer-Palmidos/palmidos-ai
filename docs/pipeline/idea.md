# Project Idea: Palmidos AI Landing Page

## Description
Create a landing page for Palmidos AI - an AI agents software development agency.

## Core Value Proposition
Palmidos AI specializes in building AI agents and automation solutions for businesses. The landing page should communicate expertise in AI development, showcase services, and drive lead generation.

## Target Audience
- Businesses looking to automate workflows with AI
- Companies seeking custom AI agent development
- Startups wanting to integrate AI into their products

## Key Sections Needed
1. Hero section with compelling headline
2. Services overview (AI agents, automation, consulting)
3. Case studies or portfolio showcase
4. About the agency
5. Contact/CTA section
6. Footer with social links

## Design Direction
- Modern, professional aesthetic
- Tech-forward but approachable
- Clean typography and spacing
- Dark/light mode consideration

## Technical Stack
- Next.js (React)
- Tailwind CSS
- Vercel deployment
- Neon PostgreSQL (for contact form submissions)
